# Distributed Training in SageMaker Unified Studio

This example demonstrates how to run distributed training jobs from Amazon SageMaker Unified Studio.

## Project Structure

```
distributed_training_sm_unified_studio/
├── scripts/
│   ├── train.py            # Training script
│   └── requirements.txt    # Python dependencies
├── distributed training in unified studio.ipynb  # Main notebook
└── README.md               # This file
```

## Quick Start

1. Open the `distributed training in unified studio.ipynb` notebook in SageMaker Unified Studio
2. Follow the instructions in the notebook to configure and launch the distributed training job
